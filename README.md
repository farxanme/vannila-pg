# Vanilla Payment Gateway

پروژه درگاه پرداخت با استفاده از HTML, CSS و Vanilla JavaScript

## ویژگی‌ها

- سازگار با مرورگرهای مدرن و IE 10
- کاملاً ریسپانسیو و چند پلتفرمی
- دسترسی‌پذیری و پشتیبانی از gesture و کیبورد
- چند زبانه (فارسی، انگلیسی، ترکی، عربی، روسی)
- کامپوننت‌های قابل استفاده مجدد
- سیستم اعتبارسنجی فرم پیشرفته

## نصب و راه‌اندازی

```bash
npm install
```

## توسعه

```bash
npm run dev
```

## بیلد برای تولید

```bash
npm run build
```
